# AgniUI

React UI Component for Agranara projects

## Installation

To install AgniUI:

```
$ yarn add @agnijs/agni-ui

# OR

$ npm install @agnijs/agni-ui


```

## Usage

```javascript
import { Button } from '@agnijs/agni-ui';

const YourCustomComponent = () => {
  return <Button>Your Text</Button>;
};
```
